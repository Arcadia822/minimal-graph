确定性出图流水线

从用户一句话，到结构化源文件、可读图像、检查报告和派生版本。

01 INPUT

真实需求

把模糊描述压缩成目标、受众、尺寸和交付物。

02 STRUCTURE

结构规划

先定阅读顺序、节点关系、泳道和信息密度。

03 SOURCE

可维护源文件

HTML、SVG、Mermaid、DOT 都作为可复现资产保存。

04 RENDER

真实渲染

用浏览器或矢量渲染器导出 PNG / SVG。

05 INSPECT

质量检查

检查字体、裁切、重叠、对齐和移动端可读性。

06 OUTPUT

归档交付

主图、源文件、派生版本和检查记录一起保留。

复杂流程图能力

  • 多泳道、分支、回路和依赖关系可控。
  • 结构语法和最终视觉可以分开维护。
Mermaid Graphviz BPMN

可视化图表能力

  • 柱状图、折线图、对比图和流程指标同屏展示。
  • 标签不进位图随机生成，数值和文字保持可编辑。
数据标签 图例 轴线

AI 插画嵌入能力

  • 插画作为透明背景物件嵌入，不替代标题和标注文字。
  • 适合封面、文章配图和说明型信息图的局部视觉补强。
透明 PNG 遮罩 留白

版式与检查能力

  • 使用网格、自动换行和固定边界减少人工坐标错误。
  • 截图后再检查，而不是只相信源文件看起来合理。
不重叠 不裁切 可读
源文件优先 HTML / SVG / 图表语法与 PNG 一起归档，后续改版不从截图猜测。
文字可控 标题、节点、标签、注释都由确定性文本排版，避免生成式乱码。
渠道解耦 核心视觉先稳定，再为 README、文章、小红书或公众号派生版本。
检查闭环 以最终渲染图为准检查裁切、重叠、字号、边距和阅读顺序。
