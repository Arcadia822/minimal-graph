# Xiaohongshu Card Template

Use this when producing reusable Xiaohongshu cards.

## Review gates

1. Storyboard review first: title, note body, and page-by-page content.
2. Image review second: send each generated page as an attachment.
3. Publish only after explicit confirmation.

Do not generate or publish before the storyboard is approved.

## Canvas

- Default size: 1242×1660 (3:4).
- Use Maple font for all visible text.
- White background, black text, black title block or border system.
- Use a rounded outer border for the reusable social-card template when requested.

## Default 3-page pattern

### Page 1: cover

- Title only.
- Minimal supporting text, preferably none.
- Footer area is reserved but hidden: no author, no page number visible.

### Page 2: workflow / logic

- Describe the stage logic in human terms.
- Do not list skill names unless the user explicitly asks.
- Use larger text than WeChat diagrams; phone readability wins over density.

### Page 3: skills / sources

- Title: `这次用到的 Skills` unless user says otherwise.
- Include GitHub/source addresses for public skills.
- Do not list private/user-made skills in public-facing images unless explicitly approved.
- Do not include `brave-search` by default unless the user asks.

## Footer

- Page 1: reserve footer space but show nothing.
- Page 2+: right footer `Author | page/total`, e.g. `Arcadia & Luna | 2/3`.
- Left footer remains empty by default.

## Copy style

- Do not add hashtags by default.
- Avoid AI-ish contrast patterns like “不是……而是……”.
- State what happened directly.
